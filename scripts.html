Fertige Skripte.
Direkt einsatzbereit.

Geprüfte PowerShell-Skripte für Windows-Administratoren. Beschreibung lesen, Parameter anpassen, herunterladen.

📋
Get-SystemInventory.ps1
Vollständiges System-Inventar: Hardware, OS, Netzwerk, installierte Software.
inventory system info
⬇️ Herunterladen
👤
Get-LocalAdmins.ps1
Alle lokalen Administratoren auf dem System auflisten. Gut für Security-Audits.
admin security audit
⬇️ Herunterladen
🌐
Test-NetworkConnectivity.ps1
Netzwerkkonnektivität zu mehreren Zielen testen. DNS, Gateway, Internet, interne Server.
network diag quick
⬇️ Herunterladen
📦
Get-InstalledSoftware.ps1
Alle installierten Programme aus 32-Bit und 64-Bit Registry exportieren. CSV-Export möglich.
inventory software audit
⬇️ Herunterladen
⚡
Set-PowerPlan-Win11.ps1
Windows 11 Energieplan optimieren. Höchstleistung aktivieren, Standby deaktivieren, WLAN-Adapter konfigurieren.
power windows11 performance
⬇️ Herunterladen
📋
Get-EventLogCollector-Client.ps1
Event Logs eines Windows-Clients sammeln und als JSON exportieren. Für den Log Analyzer gedacht. Zeitraum interaktiv wählbar.
eventlog client diag
⬇️ Herunterladen
🖥️
Get-EventLogCollector-Server.ps1
Event Logs eines Windows Servers sammeln. Erkennt installierte Rollen automatisch (AD, DNS, DHCP, RDS, Hyper-V). JSON-Export für Log Analyzer.
eventlog server diag
⬇️ Herunterladen
