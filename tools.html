📲 Als App installieren (PWA)

support.sheet kann als Progressive Web App (PWA) auf dem Gerät installiert werden – ohne App Store, direkt aus dem Browser. Funktioniert offline, öffnet wie eine native App.

🖥️
Desktop (Chrome / Edge)
1. Seite in Chrome oder Edge öffnen
2. Adressleiste → Install-Icon (⊕) klicken
3. „Installieren" bestätigen
📱
Android (Chrome)
1. Seite in Chrome öffnen
2. Menü (⋮) → „Zum Startbildschirm"
3. Namen bestätigen und hinzufügen
🍎
iPhone / iPad (Safari)
1. Seite in Safari öffnen
2. Teilen-Symbol → „Zum Home-Bildschirm"
3. Namen bestätigen und hinzufügen

📶 Offline-Cache

Der Service Worker cached alle Seiten und Daten automatisch. Die App funktioniert vollständig ohne Internetverbindung – solange der Cache aktuell ist.
