📲 Als App installieren (PWA)

support.sheet als Progressive Web App installieren – ohne App Store, direkt aus dem Browser. Funktioniert offline, öffnet wie eine native App.

🖥️
Desktop (Chrome / Edge)
1. Adressleiste → Install-Icon (⊕) klicken
2. „Installieren" bestätigen
📱
Android (Chrome)
1. Menü (⋮) → „Zum Startbildschirm"
2. Namen bestätigen
🍎
iPhone / iPad (Safari)
1. Teilen-Symbol → „Zum Home-Bildschirm"
2. Namen bestätigen
